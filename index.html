O jogo da memória é um jogo que pode ser jogado para aumentar a concentração e a memória. A jogada é quando um jogador vira duas cartas e coloca-as para cima, se o jogador virar duas cartas que não correspondem, ambos as cartas devem ser virados para baixo novamente no mesmo local.o objetivo é identificar todos os pares no menor período de tempo possível.
